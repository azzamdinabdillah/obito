📋 Project Overview

Obito React adalah platform pembelajaran online yang dibangun dengan React, TypeScript, dan Vite. Project ini menawarkan berbagai kursus dalam kategori Programming, Networking, Digital Marketing, Product Design, dan Entrepreneurship.

🎯 Fitur Utama:

  • Katalog kursus dengan 50+ kursus
  • Sistem autentikasi (Sign In/Sign Up)
  • Dashboard pembelajaran
  • Sistem subscription
  • Responsive design dengan Tailwind CSS
  • TypeScript untuk type safety

🛠️ Tech Stack

  • React 19.1.0
  • TypeScript 5.8.3
  • Vite 7.0.4
  • Tailwind CSS 4.1.11
  • React Router DOM 7.7.1

📦 Dependencies

  • @radix-ui/react-accordion
  • react-tabs
  • highlight.js
  • ESLint

⚙️ Setup & Installation

Prerequisites

  • Node.js (versi 18 atau lebih baru)
  • npm atau yarn
  • Git

Installation Steps

# Clone repository git clone [repository-url] cd obito-react # Install dependencies npm install # Start development server npm run dev # Build for production npm run build # Preview production build npm run preview # Run linting npm run lint
✅ Success! Project akan berjalan di http://localhost:5173

Environment Setup

Project ini menggunakan Vite sebagai build tool dengan konfigurasi berikut:

// vite.config.ts import { defineConfig } from 'vite' import react from '@vitejs/plugin-react' import tailwindcss from '@tailwindcss/vite' export default defineConfig({ plugins: [react(), tailwindcss()], })

🧩 Components

Core Components

NavbarDashboard NavbarDashboard.tsx Navigation bar untuk authenticated users -

Component Usage Example

import Button from "../components/Button"; // Button dengan variant green <Button variant="green" className="w-full"> Get Started </Button> // Button dengan variant outline <Button variant="outline" onClick={handleClick}> Sign In </Button>

Creating New Components

// Template untuk component baru import React from "react"; interface ComponentNameProps { // Define your props here title: string; onClick?: () => void; } const ComponentName: React.FC<ComponentNameProps> = ({ title, onClick }) => { return ( <div className="component-container"> <h3>{title}</h3> <button onClick={onClick}>Click me</button> </div> ); }; export default ComponentName;

🎨 Styling & CSS

Tailwind CSS Configuration

Project menggunakan Tailwind CSS v4 dengan custom theme variables:

@theme { --color-obito-black: #0a0723; --color-obito-green: #2f6a62; --color-obito-grey: #eaecee; --color-obito-red: #ef372b; --color-obito-text-grey: #87898c; --color-obito-light-green: #e0eae8; --color-obito-light-red: #ffe3e1; --color-obito-text-secondary: #6a6f7c; }

Custom CSS Classes

// Utility classes .flex-center { @apply flex justify-center items-center; } .bg-custom-gradient { background: linear-gradient(90deg, #a531ff 0%, #ff1fa5 100%); } .container-max-w-1280 { @apply max-w-[1280px] mx-auto; } .container-padding-x { @apply px-4 md:px-8 lg:px-[75px]; } .container-padding-y { @apply py-4 md:py-8 lg:py-[39px]; }

Responsive Design

Project menggunakan Tailwind breakpoints untuk responsive design:

  • sm: 640px dan ke atas
  • md: 768px dan ke atas
  • lg: 1024px dan ke atas
  • xl: 1280px dan ke atas
// Contoh responsive classes <div className="text-sm md:text-base lg:text-lg"> Responsive text </div> <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3"> Responsive grid </div>

📊 Data Management

Data Structure

Project menggunakan data statis yang disimpan dalam file TypeScript:

Course Interface

export interface Course { id: string; title: string; category: string; thumbnail: string; rating: number; lessons: number; status: string; categoryIcon: string; lessonsIcon: string; statusIcon: string; ratingIcon: string; }

Subscription Interface

interface SubscriptionData { id: number; name: string; duration: string; icon: string; price: string; startedAt: string; status: "ACTIVE" | "EXPIRED" | "PENDING"; detailsUrl: string; }

Data Files

  • courses.ts: Data 50+ kursus dalam 5 kategori
  • testimonials.ts: Data testimonial users
  • roadmap.ts: Data learning roadmap
  • courseLearning.tsx: Data pembelajaran kursus

Adding New Data
