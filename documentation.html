📋 Project Overview

Obito React is a UI template only for an online learning platform built with React, TypeScript, and Vite. This project showcases various UI components and pages for Programming, Networking, Digital Marketing, Product Design, and Entrepreneurship categories.

⚠️ Important Notice:

This is a UI template only - no backend functionality is included.

  • All data is static/mock data
  • No real authentication system
  • No actual payment processing
  • No database integration
  • Forms don't actually submit data

🎯 UI Template Features:

  • Complete UI mockup with 50+ course displays
  • Authentication UI pages (Sign In/Sign Up)
  • Learning dashboard interface
  • Subscription management UI
  • Responsive design with Tailwind CSS
  • TypeScript for type safety
  • Ready-to-use component library

🛠️ Tech Stack

  • React 19.1.0
  • TypeScript 5.8.3
  • Vite 7.0.4
  • Tailwind CSS 4.1.11
  • React Router DOM 7.7.1

📦 Dependencies

  • @radix-ui/react-accordion
  • react-tabs
  • highlight.js
  • ESLint

⚙️ Setup & Installation

Prerequisites

  • Node.js (versi 18 atau lebih baru)
  • npm atau yarn
  • Git

Installation Steps

# Ekstrak file ZIP
unzip obito.zip
cd obito

# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build

# Preview hasil build production
npm run preview

# Jalankan linting
npm run lint
✅ Success! Project will run at http://localhost:5173

  );

🛣️ Routing

Route Configuration

// src/App.tsx
<Routes>
  <Route path="/" element={<Homepage />} />
  
  <Route path="/pricing">
    <Route index element={<Pricing />} />
    <Route path="checkout">
      <Route index element={<Checkout />} />
      <Route path="success" element={<SuccessCheckout />} />
    </Route>
  </Route>
  
  <Route path="/course">
    <Route path="catalog" element={<CourseCatalog />} />
    <Route path="learning" element={<CourseLearning />} />
    <Route path="details" element={<CourseDetails />} />
    <Route path="search" element={<SearchCourse />} />
    <Route path="learning-finished" element={<LearningFinished />} />
  </Route>
  
  <Route path="/signup" element={<SignUp />} />
  <Route path="/signin" element={<SignIn />} />
  
  <Route path="/subscription">
    <Route index element={<Subscription />} />
    <Route path="details" element={<SubscriptionDetails />} />
  </Route>
</Routes>
